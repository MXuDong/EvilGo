# Evil golang 版

EvilJS 的 狗朗 实现。功能不太丰富。。。。。有时间慢慢加吧。

功能：
- 有百分之1的概率可能会将 Golang 时间获取冻结掉为格林威治0时时间。
- 有百分之1的概率 fmt.Println 不受控制，胡乱打印。

## Usage ?

显示调用 Evil() 即可

使用时需要关闭内联优化：

```
-gcflags="-l"
```

## 修改配置？

当然可以，通过 EvilOptions 可以设置触发概率，影响值，以及影响的持续时间。
